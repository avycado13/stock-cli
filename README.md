# stock-cli ![Crates.io Version](https://img.shields.io/crates/v/stock-cli)


A CLI to view stocks with graphing. THe first arg is the ticker (GOOG, AMZN, IBM, etc). Defaults to IBM. Second arg is number of days to show from today. defaults to 5.

## Install
[Crates.io link](https://crates.io/crates/stock-cli)
```bash
cargo install stock-cli
```
